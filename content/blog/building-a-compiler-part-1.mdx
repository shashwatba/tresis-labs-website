## Starting with the lexer

The first stage of any compiler is **lexing** (or tokenizing): taking raw source text and
breaking it into meaningful tokens like keywords, identifiers, and operators.

```python
# Example token output for: x = 1 + 2
[
  Token(IDENT, "x"),
  Token(ASSIGN, "="),
  Token(INT, "1"),
  Token(PLUS, "+"),
  Token(INT, "2"),
]
```
